# [3주차] 클로드 코드 시작하기: 설치부터 손글씨 인식, 그리고 CLAUDE.md까지

## 🎯 1. 수업 핵심 역량
* **바이브 코딩**: AI 코딩 파트너를 활용하여 자연어 프롬프트로 빠르게 개발하는 방식이다.
* **컴퓨팅 사고**: 문제를 분해하고, 패턴을 인식하며, 추상화하여 알고리즘적으로 해결하는 능력이다.
* **프로젝트 관리**: 요구사항을 정의하고, 일정 계획과 중간 보고를 거쳐 최종 발표에 이르는 과정이다.

## 💡 2. 클로드 코드(Claude Code)와 개발 환경
클로드 코드는 웹 브라우저가 아닌 내 컴퓨터의 터미널(명령 프롬프트)에서 실행되는 AI 코딩 도구이다. 실제 파일을 생성하고 실행하며 복잡한 프로젝트를 관리하는 데 최적화되어 있다.

### 클로드 웹 vs 클로드 코드 비교 
| 구분 | 클로드 웹 (claude.ai) | 클로드 코드 (Claude Code) |
| :--- | :--- | :--- |
| **실행 환경** | 웹 브라우저 | 내 컴퓨터의 터미널 |
| **작업 방식** | Artifacts에서 미리보기 | 실제 파일 생성 및 실행 |
| **활용 범위** | 간단한 코드, 문서, 웹페이지 | 복잡한 프로그램, 프로젝트 관리 |
| **파일 접근** | 제한적 | 내 컴퓨터 파일에 직접 접근 |

### 설치 전 필수 준비물 
1. **Node.js**: 공식 사이트에서 LTS 버전을 설치해야 한다. (확인: `node --version`)
2. **Git**: 기본 설정으로 설치를 진행한다. (확인: `git --version`)
3. **터미널**: 텍스트 명령어로 컴퓨터를 조작하며, `dir`(목록), `cd`(이동), `mkdir`(폴더 생성) 명령어를 기본으로 사용한다.

## 🛠️ 3. 실습: MNIST 손글씨 인식 및 프로젝트 확장
딥러닝 입문 표준 데이터셋인 MNIST를 활용하여 손글씨 인식 프로그램을 만들고, 이를 웹 버전으로 확장하는 과정을 거친다.

### 1단계: 클로드 코드 설치 및 실행
* 터미널에서 `npm install -g @anthropic-ai/claude-code` 명령어로 설치한다.
* 프로젝트 폴더에서 `claude`를 입력하여 실행하고 테마 설정 및 로그인을 완료한다.

### 2단계: 손글씨 인식 프로그램 제작
* **프롬프트**: "손글씨로 숫자를 입력하면 인식하는 코드를 만들고 실행해 줘"라고 요청한다.
* **자동화**: 클로드 코드는 코드 생성, 패키지 설치, 실행을 한 번에 처리한다.
* **독립 실행**: 배치 파일(`.bat`) 생성을 요청하면 터미널 없이 더블 클릭만으로 프로그램을 실행할 수 있다.

### 3단계: CLAUDE.md를 활용한 관리 (추상화의 실전)
* **개념**: 프로젝트의 목적, 기술 스택, 코딩 규칙 등을 담은 "프로젝트 매뉴얼"이다.
* **자동 생성**: `/init` 명령어를 입력하면 폴더 구조를 분석하여 자동으로 파일을 만든다.
* **규칙 추가**: 대화 중 `#` 키를 누르면 특정 규칙(예: 모든 파일에 날짜 주석 추가)을 빠르게 `CLAUDE.md`에 반영할 수 있다.
* **계층적 구조**: 루트 폴더와 하위 폴더(web_version 등)에 각각 `CLAUDE.md`를 두어 폴더별 특화 규칙을 적용한다.
