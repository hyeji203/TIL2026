# 1️⃣ 핵심 개념

## 📌 할루시네이션 (Hallucination)
- AI가 **사실이 아닌 내용을 그럴듯하게 생성하는 현상**

👉 발생 이유  
- 확률 기반 문장 생성  
- 불완전한 정보 추론  

👉 해결 방법  
✔ 공식 자료와 교차 검증  
✔ 명확한 기준 제시  

💡 AI 답변은 반드시 검증 필요

---

## 📌 컴퓨팅 사고

| 사고 | 의미 |
|------|------|
| 분해 | 문제를 작은 단위로 나눔 |
| 패턴 인식 | 반복 구조 찾기 |
| 추상화 | 핵심만 정리 |
| 알고리즘 | 순서대로 해결 |

👉 이번 핵심: **패턴 인식 → 자동화**

# 4️⃣ 할루시네이션 해결

## 📌 문제 사례
- "세계에서 가장 긴 강" → 논란 존재

---

## 📌 해결 방법
✔ 기준 명확화 (예: 전통 기준)  
✔ 교차 검증  

---

## 📌 가이드라인

1. 정답이 하나인지 확인  
2. 최상급 표현 기준 명시  
3. 시간/범위 명확화  
4. 2개 이상 출처 검증  

---

## 📌 CLAUDE.md 저장

~~~bash
/memory
~~~

👉 프로젝트 규칙 저장

# 5️⃣ 컨텍스트 관리

## 📌 /clear vs /compact

| 명령어 | 기능 |
|--------|------|
| /clear | 완전 초기화 |
| /compact | 요약 유지 |

---

## 📌 사용 상황
- 새 작업 → /clear  
- 계속 작업 → /compact  

---
