👉 쉽게 말하면  
"프로그램 ↔ AI 연결 다리" :contentReference[oaicite:0]{index=0}

## 📌 동작 흐름

~~~
사용자 → 앱 → API → AI 모델
                   ↓
사용자 ← 앱 ← API ← 결과 반환
~~~

---

## 📌 특징

✔ 복잡한 AI를 직접 만들 필요 없음  
✔ 외부 AI 기능 활용 가능
