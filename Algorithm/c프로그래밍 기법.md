# 1️⃣ 배열 (Array)

## 📌 배열 개념
- 같은 자료형 데이터를 연속된 메모리 공간에 저장
- 인덱스(index)는 0부터 시작
- 1차원, 2차원, 3차원 등 다양한 형태 존재

---

## 📌 1차원 배열 선언
~~~c
int arr[5];
~~~

- 자료형 배열이름[크기]
- 배열 크기만큼 메모리 할당

---

## 📌 배열 초기화
~~~c
int arr[5] = {1,2,3,4,5};
~~~

- 초기값이 부족하면 나머지는 0으로 채워짐

~~~c
int arr[] = {1,2,3};
~~~

---

## 📌 문자 배열 (문자열)
~~~c
char str[10] = "hello";
~~~

- 문자열은 char 배열로 저장
- 마지막에 '\0' 포함됨

---

## 📌 다차원 배열
~~~c
int arr[2][3];
~~~

- 배열 안에 배열 구조
- 메모리는 연속적으로 저장됨

---

# 2️⃣ 포인터 (Pointer)

## 📌 포인터 개념
- 메모리 주소를 저장하는 변수
- 다른 변수의 위치를 가리킴

---

## 📌 포인터 선언
~~~c
int *ptr;
~~~

---

## 📌 주요 연산자

### ✔ 주소 연산자 (&)
~~~c
ptr = &a;
~~~

- 변수의 주소 반환

### ✔ 참조 연산자 (*)
~~~c
*ptr = 10;
~~~

- 해당 주소의 값 접근

---

## 📌 포인터 사용 예
~~~c
int a = 10;
int *ptr = &a;

printf("%d", *ptr); // 10
~~~

---

## 📌 포인터 초기화 방법

1. 변수 주소 사용
~~~c
int *ptr = &a;
~~~

2. 동적 할당
~~~c
ptr = (int*)malloc(sizeof(int));
~~~

3. 배열과 연결
~~~c
int arr[10];
int *ptr = arr;
~~~

---

## 📌 포인터 배열
~~~c
char *arr[3] = {"A", "B", "C"};
~~~

---

## 📌 이중 포인터
~~~c
int **ptr;
~~~

---

# 3️⃣ 구조체 (Struct)

## 📌 구조체 개념
- 서로 다른 자료형을 하나로 묶은 자료형
- 복잡한 데이터 표현 가능

---

## 📌 구조체 선언
~~~c
struct student {
    char name[10];
    int age;
};
~~~

---

## 📌 구조체 변수 선언
~~~c
struct student s1;
~~~

---

## 📌 구조체 초기화
~~~c
struct student s1 = {"Kim", 20};
~~~

---

## 📌 구조체 접근

### ✔ 점 연산자 (.)
~~~c
s1.age = 20;
~~~

### ✔ 화살표 연산자 (->)
~~~c
ptr->age = 20;
~~~

---

## 📌 구조체 특징
~~~c
s2 = s1;
~~~

- 구조체 전체 복사 가능

---

# 4️⃣ 재귀호출 (Recursion)

## 📌 개념
- 함수가 자기 자신을 호출하는 방식
- 문제를 작은 단위로 나누어 해결

---

## 📌 핵심 요소
- Base Case (종료 조건) 필수
- 없으면 무한 반복 발생

---

## 📌 팩토리얼 예제
~~~c
int fact(int n) {
    if (n == 1) return 1;
    return n * fact(n-1);
}
~~~

---

## 📌 특징
- 코드가 간결함
- 이해하기 쉬움
- 스택 사용량 증가

---

# 🔥 핵심 정리 (시험 대비)

✔ 배열 → 연속된 메모리, 인덱스 0부터  
✔ 포인터 → 주소 저장, *, & 중요  
✔ 구조체 → 다양한 자료형 묶기  
✔ 재귀 → 자기 자신 호출 + 종료조건 필수  

---

# 💡 한줄 요약
👉 자료구조 구현 = 배열 + 포인터 + 구조체 + 재귀
